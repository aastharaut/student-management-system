# Student Management System

A full-stack Student Management System built using React, TypeScript, Node.js, Express, and PostgreSQL.

## Live Demo

Frontend: https://student-management-system-nine-lovat.vercel.app/
Backend: https://student-management-system-7qhw.vercel.app/

## Features

- Add students
- Update student details
- Delete students
- Role-based authentication
- Profile picture upload
- Dashboard analytics

## Tech Stack

Frontend

- React
- Redux
- TypeScript
- Tailwind CSS

Backend

- Node.js
- Express
- Sequelize

## Database

The project uses PostgreSQL hosted on Neon.

Neon provides a serverless PostgreSQL database used to store:

- User accounts
- Student records
- Profile picture URLs
